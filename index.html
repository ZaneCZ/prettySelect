Work in progress

PrettySelect is a simple jQuery plugin to turn your old select boxes into pretty ones, allowing you to pass in data dynamically.

Basic usage

Include js and css files

<head>
    <link rel="stylesheet" href="bootstrap.min.css">
    <link rel="stylesheet" href="prettySelect.css">
    <script src="jquery.min.js"></script>
    <script src="prettySelect.js"></script>
</head>

And call prettySelect() once the page is loaded.

$("select").prettySelect();

Default values

var settings = {
    action: "create",
    wrapperTemplate: "<div class='form-control'>",
    labelTemplate: "<span class='label label-primary'>",
    listTemplate: "<div class='selectList'>",
    listItemTemplate: "<div>",
    itemRemoveTemplate: "<span>&times;</span>",
    searchEnabled: true,
    searchDebounce: 100,
    valuesHandler: null,
    listFilterHandler: filterList
};

action ["create", "destroy" or "values"]

create initiates prettySelect

destroy destroys prettySelect object and shows original select box

options returns list of all currently available options

wrapperTemplate [html string or jQuery object]

has class prettySelect

labelTemplate [html string or jQuery object]

has class selectLabel

listTemplate [html string or jQuery object]

has class selectList

listItemTemplate [html string or jQuery object]

has class selectListItem

itemRemoveTemplate [html string or jQuery object]

has class close

searchEnabled [boolean]

If set to true, search bar is shown, else all available options are shown.

searchDebounce [integer]

It is highly recommended to set this above 0 if you load data through ajax.

valuesHandler [function(wrapper, searchText)]

First argument is main object of prettySelect, second argument is content of searchbar. This function should return object in format {value:text}. It is possible to load values through AJAX.

listFilterHandler [function(searchText, value)]

First argument is content of searchbar, second argument is value of possible options. Value gets shown in option list if filter function returns true.

Get options

if any of prettySelects exists
var options = $("select").prettySelect();
else

var options = $("select").prettySelect("options");

or

var options = $("select").prettySelect({
    action: "options"
});

Get values

$("select").on("change",function(){
    var value = $(this).val();
});
